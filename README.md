# Google Sheets Habit Tracker

A simple and interactive Habit Tracker built in Google Sheets to track daily habits and visualize progress.

## Overview
This project is a personal productivity tool designed to help users track their daily habits throughout the week. It uses checkboxes, formulas, and sparkline charts to automatically calculate habit completion and display progress.

The tracker helps users stay consistent and monitor their habits visually.

## Features
- Daily habit tracking using checkboxes
- Automatic progress calculation
- Weekly progress visualization with SPARKLINE charts
- Simple and clean interface
- Easy to customize and add new habits

## Tools Used
- Google Sheets
- Formulas (COUNTIF, SPARKLINE)
- Checkbox automation
- Progress visualization

## How to Use

1. Open the Google Sheets file.
2. Click **File → Make a copy** to save it to your own Google Drive.
3. Add your habits to the habit column.
4. Mark completed habits using checkboxes each day.
5. The progress bar will update automatically.

## Future Improvements
- Monthly statistics dashboard
